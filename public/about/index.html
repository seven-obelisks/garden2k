About

Garden2k is an open source static site template built with Hugo and Decap CMS. It is designed for non-profits and features a built-in event calendar system. Some features:

  • Homepage banner can be edited via the CMS
  • Events automatically generate in an .ICS format so attendees can easily add an event reminder to their personal calendar
