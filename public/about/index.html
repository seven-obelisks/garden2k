About

Garden2k is an open source static site template built with Hugo and Decap CMS on top of the Ananke theme. It is designed for event-heavy organizations that need a low-cost static site. Some features:

  • Free to deploy and maintain on Netlify
  • Homepage slider banner can be edited via the CMS, including adding additional slides
  • Events automatically generate an .ICS file so attendees can easily add an event reminder to their calendar
